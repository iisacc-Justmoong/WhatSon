# `src/app/models/editor/tags/ContentsEditorTagInsertionController.cpp`

## Responsibility

Implements shortcut-to-tag resolution and delegates RAW mutation payload construction to
`ContentsEditorTagMutationBuilder`.

## Current Behavior

- `tagNameForShortcutKey(...)` resolves `B`, `I`, `U`, and `E` shortcut keys to `bold`, `italic`, `underline`, and
  `highlight`.
- `tagNameForBodyShortcutKey(...)` resolves explicit body-tag shortcuts to `agenda`, `callout`, or `break`, including
  macOS Option-modified key variants such as `ç`/`Ç` for the callout command.
- `normalizedTagName(...)`, `buildTagInsertionPayload(...)`, and `buildWrappedTagInsertionPayload(...)` forward to the
  owned `ContentsEditorTagMutationBuilder` so callers that still depend on the controller API remain compatible.
- The controller re-emits `tagInsertionPayloadBuilt(...)` from its builder for existing signal observers.

## Boundary

The controller does not contain the tag-generation algorithm. Formatting tags and transparent paired body tags are
generated by `ContentsEditorTagMutationBuilder`, which keeps tag creation usable from menus, toolbar actions, command
palettes, and other future non-shortcut surfaces.
