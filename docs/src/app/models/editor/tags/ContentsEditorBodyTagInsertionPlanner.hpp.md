# `src/app/models/editor/tags/ContentsEditorBodyTagInsertionPlanner.hpp`

## Responsibility
Declares the C++ RAW body-tag insertion planner used by editor input command paths.

## Contract
- Lives under `editor/tags` because it owns generated editor-body tags, not visual rendering.
- Accepts agenda and callout backend objects through QObject properties so QML hosts do not build canonical tag text
  themselves.
- Exposes invokable payload builders for:
  - structured shortcuts: agenda, callout, and break
  - selected-range callout wrapping with explicit `<callout>...</callout>` source tags
  - generic raw tag insertion, including resource tags generated by the resource tag controller
- Returned payloads contain `nextSourceText`, `sourceOffset`, and `insertedSourceText`; QML hosts apply the payload
  and then let parser/renderer projection rematerialize the editor.

## Boundary
- This class does not persist files and does not render tags.
- Inline style range mutation remains under editor formatting; this planner handles body-level generated tags and raw
  tag insertion payloads.
